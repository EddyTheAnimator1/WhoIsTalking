# Installation
Drop WhoIsTalking.dll into GorillaTag/BepInEx/plugins/

Run Gorilla Tag once to generate/update WhoIsTalking.cfg

Tweak settings in %GAMEDIR%/BepInEx/config/WhoIsTalking.cfg and enjoy!
